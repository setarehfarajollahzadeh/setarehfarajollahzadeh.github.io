Academic website template

Files:
- index.html: homepage
- research.html: research page
- assets/style.css: website design

How to use on GitHub Pages:
1. Upload index.html, research.html, and the assets folder to the root of your repository.
2. Replace assets/profile.jpg with your own photo and keep the name profile.jpg.
3. Upload your CV as cv.pdf in the root folder.
4. Commit the changes.
5. Wait 1-2 minutes for GitHub Pages to update.
